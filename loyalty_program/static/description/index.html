Loyalty Program for Sale Odoo Apps

Loyalty program for Sale odoo apps is used to give Sale loyalty redemption points for every Sale to your customers. Customer can also redeem this loyalty points for other Sale. Every single Sale records the Loyalty Rewards based on configuration setup on there backend and those rewards will be redeemed on Sale easily. Reward redeems visible on applied Sale order so it will be helpful for see history of the reward point redemption. We can also view all loyalty point which will be calculate based on history.

Features

Loyalty Point Configuration

Apply loyalty point based on loyalty configuration rule.

Earns loyalty points

Customer can earns loyalty points for every sale order.

Define Loyalty Points

Define Loyalty based on fixed or variant pourcent of sale order amount.

Redeem loyalty points

Customer can redeem loyalty points as a discount for other sale order.

Calculate Partner's Loyalty Points based on History

User can view partner's loyalty which is calculated based on partner's history.

Sale Loyalty Configuration

There are two options for loyalty configuration (Fix or variant).

So we can create many loyalty program and choose wich one we want to use in sale order.

Sale Loyalty Level

We can create many level for cutomer accord to his points.

Loyalty history

Sale order Loyalty

Point are calculated according to configuration and we can see it on sale order.
